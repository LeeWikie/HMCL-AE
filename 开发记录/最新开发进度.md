## 二、已完成并落地的能力（截至 v0.2.1-beta 及开发中）

**Agent 内核**

- 计划模式 `/plan`、上下文压缩 `/compact`、可见 TODO 卡片

- 上下文管理：工具结果封顶、超预算驱逐旧结果、read 分页（#51）

- 一键提供商预设（12+ 国内外）、多提供商加载/测试三态树、API Key 掩码
- Token/用量 + 花费估算显示、思考过程(reasoning)折叠渲染（#53）

**工具能力（约 70+）**
- 版本/加载器安装、模组搜索安装更新、实例管理、世界/数据包/整合包、账号（含微软 OAuth）、Java 管理、游戏选项、系统信息、剪贴板、SLP ping
- **存档 NBT 编辑套件**（旗舰）：read/get/set NBT、离线 UUID、跨存档搬玩家数据
- **世界备份引擎**、MCP 工具真执行（#57）、联网搜索（Bocha/Zhipu 国内直连）、OCR

**安全 / 合规**
- 两级确认（普通危险 + 红色高危）、危险命令永远硬拦（含编码命令扫描）、回收站删除
- 记忆脱敏+去重（#55）、prompt 注入防御（#63）、成本护栏（每日上限+预警，#62）、隐私同意告知（#61）
