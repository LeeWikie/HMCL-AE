# HMCL-AE 0.1.0-alpha

Agent 逻辑重写后的首个 alpha 版本,版本序列自本版重置。以下为本版本的变更内容。

## 重要更新

### AI 助手 — Agent 逻辑重构
- 统一护栏架构:工具调用拒绝态短路、runtime-guard 通道、结构化 BlockReason,消除"假成功"回执。
- 错误处理统一为错误信封(ErrorEnvelope):失败回执携带原因、作用域内候选、后续动作。
- 文件工具契约化:读取台账(ReadLedger)、写后校验、替换链;写入前强制读取。
- 各工具域(实例 / 世界 / 存档备份 / 模组 / 资源包与光影 / NBT / 搜索 / MCP)失败态统一枚举真实候选名单,不再静默回落或裸串错误。
- 资源占用统一守卫(GameResourceGuard);模组启停、删除、更新收编入状态机;世界备份 / 恢复 / NBT 写入原子性保护;删除实例前运行检测。

### AI 助手 — 诊断 Trace 基建
- 新增独立 append-only JSONL,逐轮记录 request / response / tool 调用的完整内容与 token 计数;可配置开关。
- 反馈上传附带 ai-trace / ui-trace / 元信息。

### AI 助手 — 模型参数库
- 内置模型元数据由 65 项扩充至 1604 项,涵盖上下文窗口、输出上限、模态、定价。
- 按模型 ID 自动匹配上下文窗口;新增大小写与 provider 前缀的保守规范化匹配。

### AI 助手 — 运行时状态传播修复(单一事实源)
- 修复下拉切换模型后请求仍使用旧模型。
- 每模型参数(温度 / 最大输出 / 推理强度 / 上下文窗口)接入请求侧;推理强度接入 OpenAI 与 Anthropic 请求。
- 审批模式、联网开关、shell / NBT / OCR 工具开关改为运行时实时生效,不再需要重启。
- 工具权限跨实例磁盘同步;文件系统工具允许根随实例切换替换。

## 变更明细

### 界面 — 对话栏
- 对话栏重构为输入框 + 底部工具栏两行布局;模型选择、思考强度、上下文用量移入工具栏。
- 上下文用量数值支持百万级(1M)格式;进度条改用原生下载条样式。
- 模型 / 思考 / 审批 / 上下文四类弹窗开合动画统一;出现位置按窗口余量自适应左右展开。
- 修复对话栏空态过高、操作栏与输入框间距过大;发送后自动滚动至底部,流式输出仅在贴近底部时跟随。

### 界面 — 设置与主题
- 设置页分区重构;高级设置精简;模型配置弹窗改固定高度滚动。
- API Key 输入改为内联掩码与可见切换。
- 暗色主题输入框、表头、代码注释配色修复;代码块改用深色框(One Dark)配色。
- 定价与成本估算相关界面归档隐藏。

### 界面 — Markdown 渲染
- 裸 URL 自动链接;国旗 emoji 换源修复;代码块横向滚动;emoji 与粗体、斜体、删除线混排保留样式。
- Markdown 字号随字号档位(CSS em)。

### AI 助手 — 工具与提示词
- 联网搜索、OCR 工具按开关条件暴露,关闭时不出现在工具列表与提示词中。
- 技能索引改为扁平结构;技能召回收敛;技能改用自适应匹配(全量技能树 + 提示 nudge)替代全文注入。
- 提示词注入选中实例的权威 Minecraft 版本与类型。
- 工具描述内嵌反模式修正(shell 使用规则、PowerShell 5.1 陷阱、模组安装去过度承诺)。

### 安全
- SSRF / DNS 加固;修复路径穿越与符号链接逃逸;锁定 Jackson 依赖 CVE 版本。

### 构建与发布
- 新增 `-PheadlessTest`,CI 与后台测试无头运行,不弹出 JavaFX 窗口。
- 发布产物改为正文内嵌更新日志与 SHA-256 校验和表格。

### 其他
- i18n 文案全量迁移(约 230 键);繁体中文文案重译。
- 移除仓库内截图目录;停止跟踪运行时状态目录。
