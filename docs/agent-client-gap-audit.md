### 结论

基础三块（对话流式 + Markdown、会话 CRUD、输入区）已相当完整，短板集中在四个方向：

1. **Agent 高级渲染**（缺口最大，也最能拉开与「普通聊天窗」的差距）：
   思考过程折叠 B1、实时 Markdown / 回合容器 B2、审批内联卡 + diff B5、任务进度卡 B7、停止续写 B8。
2. **可操作的状态反馈**：错误卡片 A13、首启引导 A14、上下文占用 C7、回到底部 B4。
3. **渲染打磨**：气泡自适应 A4、代码块语法高亮 + 一键复制（net-new）、选中复制 #37、搜索会话、时间分组。
4. **合规底座**：i18n H1、文案人话化 A12、键盘可达 C8。
