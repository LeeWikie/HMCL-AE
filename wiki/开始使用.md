# 开始使用

## 安装

1. 从 [Releases](https://github.com/LeeWikie/HMCL-AE/releases) 下载最新版。
2. **Windows**: 双击 `.exe`。
3. **Linux / macOS**: `java -jar HMCL-AE-xxx.exe`(exe 同时也是合法 jar)。

## 首次配置

1. 启动后进入左侧栏 **「AI 助手」**(空心信息图标 ⓘ)。
2. 进入 **「AI 设置」→「服务商」**,点击「添加」。
3. 选择提供商的**协议族**(OpenAI / OpenAI Reasoning / Anthropic / REST API)。
4. 填入 **API 端点 + API Key + 模型名**。
5. 保存后设置页会尝试连接并拉取该服务商的模型列表。

> 目前没有任何内置的免费模型——你必须配置自己的 API key。

## 第一条消息

配置好提供商店,直接打字就行:

- 「我装了哪些实例?」
- 「帮我装个最新版 + Fabric,再装 Sodium 及能用的附属」
- 「分析一下我最新的崩溃报告」
- 「把 xxx 实例改名为 yyy」
- 「搜一下最近 MC 1.22 的更新新闻」

AI 会自己调用工具干活。
